liquidctl executables for Windows bundle the following projects:

 - liquidctl
 - pyinstaller (bootloader)
 - python
 - docopt
 - colorlog
 - colorama
 - pyusb
 - libusb-1.0
 - cython-hidapi
 - hidapi

Applicable disclaimers, original licenses, and links for the source code of
each project are provided bellow.
